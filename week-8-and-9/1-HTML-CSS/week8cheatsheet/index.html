DBC Cheat Sheet

By: Jonathan Hinkey and Pablo Lopez Domowicz

Arrays

.empty?

a = ["cat","hat","bat"]
b = []
a.empty? => false
b.empty? => true

The .empty? function checks to see if an array has any content. It will return true when the array is empty and false when it is not.

.include?(input)

a = ["cat","hat","bat"]
a.include?("cat") => true
a.include?("mouse") => false

The .include? function checks to see if a variable is in an array. It will return true when the variable is found and false when it is not.

.push(input) and <<

a = ["cat","hat","bat"]
a.push("mouse") => a = ["cat","hat","bat","mouse"]
a << "calculator" => a = ["cat","hat","bat","mouse", "calculator"]

The .push and << function takes an element and adds it to the end of an existing array.

.pop

a = ["cat","hat","bat"]
a.pop => a = ["cat","hat"]

The .pop function removes the last element from an existing array.

.shift

a = ["cat","hat","bat"]
a.shift => a = ["cat","hat"]

The .pop function removes the last element from an existing array.

.unshift(input)

a = ["cat","hat","bat"]
a.unshift("paper") => a = ["paper","cat","hat","bat"]

The .unshift function takes and element and adds it to the beginning of an existing array.
